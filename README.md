# Video Recording Demo

This is a clean React + TypeScript project scaffolded with Vite. It demonstrates video recording using [react-media-recorder](https://www.npmjs.com/package/react-media-recorder).

## Setup

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm run dev
   ```

Open [http://localhost:5173](http://localhost:5173) to view the app.

## Features
- Start/stop video recording
- Watch the recorded video in the browser
